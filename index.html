Gesture recognition dataset intended for remote control applications development based on hand movements

About the Dataset

This dataset comprises comprehensive gesture recognition data specifically designed to facilitate the development of remote control applications that utilize hand movements. The dataset is versatile and can be applied in various domains, including but not limited to:

  • Smart Home Devices: The dataset can be employed to enhance the functionality of smart home systems. This includes controlling lights, music systems, televisions, heating, ventilation, and air conditioning (HVAC) systems, and other smart appliances. By integrating gesture recognition, users can enjoy a more intuitive and seamless interaction with their home environment, improving convenience and accessibility.
  • Human-Machine Interfaces for Industrial Applications: In industrial settings, the dataset can be used to develop advanced human-machine interfaces. These interfaces can improve the efficiency and safety of operations by allowing workers to control machinery and equipment through simple hand gestures. This application is particularly beneficial in environments where traditional control methods may be cumbersome or unsafe.

Overall, the dataset provides a robust foundation for developing innovative applications that leverage gesture recognition technology to improve user interaction and control across various fields.

Gestures Description

Dataset contains the following gestures:

  • swipe left
  • swipe right
  • double shake
  • double thumb tap
  • clockwise rotation
  • counter-clockwise rotation
  • idle
  • unknown (random) gestures

Data Volume and Diversity

  • Dataset was collected by 5 unique subjects
  • Dataset contains 109 unique sessions
  • Total length raw data is 1,320,119 rows, 220 minutes
  • Total length of centered data is 797,061 rows, 132 minutes

Data collection settings

  • Sampling rate: 100 Hz
  • Sensor data: 6-axis IMU
  • Accelerometer range: ±4g
  • Gyroscope range: ±1000dps
  • Data format: CSV
  • Data encoding: UTF-8
  • Data delimiter: comma
  • Data type: integer 16-bit
